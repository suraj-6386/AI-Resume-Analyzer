================================================================================
PROJECT FILE MANIFEST - ATS OPTIMIZER PRO
================================================================================

This document provides a complete manifest of every file in the project,

================================================================================
ROOT FILES
================================================================================

FILE: app.py
ROLE: Main Flask application orchestrator
FEATURES IMPLEMENTED:
  - Route handlers (/, /analyze, /report, /history, /editor)
  - Job description input handling
  - MySQL database integration
  - Dynamic editor analysis API (/api/analyze-editor)
  - PDF download functionality
  - Session management

FILE: config.py
ROLE: Central configuration and API settings
FEATURES IMPLEMENTED:
  - API key management (Google, OpenAI)
  - Database configuration (MySQL, SQLite)
  - Scoring weights configuration
  - Theme colors and fonts
  - Environment variable handling

FILE: requirements.txt
ROLE: Python dependencies specification
FEATURES IMPLEMENTED:
  - Flask web framework
  - AI/LLM integration packages
  - Local embeddings (sentence-transformers)
  - PDF processing libraries
  - MySQL database connectors

FILE: database.sql
ROLE: MySQL database schema
FEATURES IMPLEMENTED:
  - User profiles table
  - Analysis reports table
  - Extracted skills tracking
  - AI recommendations storage
  - Detected sections tracking
  - Contact info extraction
  - Editor sessions history
  - Improvement history tracking

FILE: README.md
ROLE: Project documentation and setup guide
FEATURES IMPLEMENTED:
  - Fresh machine setup guide
  - Feature overview
  - API endpoints documentation
  - Configuration reference
  - Usage guide

FILE: usermanual.txt
ROLE: End-user manual
FEATURES IMPLEMENTED:
  - Step-by-step setup instructions
  - API key configuration guide
  - Database setup instructions
  - Feature usage guide
  - Troubleshooting section

================================================================================
CORE MODULES
================================================================================

FILE: core/__init__.py
ROLE: Package initialization
FEATURES IMPLEMENTED: Empty marker file

FILE: core/analyzer.py
ROLE: AI-powered analysis engine
FEATURES IMPLEMENTED:
  - PDF/TXT text extraction
  - Contact info regex extraction
  - Section detection (Experience, Education, Skills, etc.)
  - Semantic analysis using embeddings (90%+ accuracy)
  - AI-powered scoring (Tone & Style, Content, Structure, Skills Match)
  - Job description matching with keyword analysis
  - Impact verb scanning
  - Skill matching and classification
  - Live editor analysis support

FILE: core/pdf_builder.py
ROLE: Editorial PDF generator
FEATURES IMPLEMENTED:
  - Professional PDF output
  - A4 page formatting
  - Styled headers and sections
